# SpotifyCodeOnPatterns
simple .png spotify code on pattern generator

# installing:
```
  git clone https://github.com/Danko118/SpotifyCodeOnPatterns ~/scop
  cd scop
  pip install -r requirements.txt
```

# usage:
```
python CodeGenV1 
```
